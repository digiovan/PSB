Files for the LTP / LTB / BI line from the LINAC2 to the PSB

The element files contain some elements (mostly quadrupoles) that are about 10% too long.
The reason is that these longer elements give better optics results when used in program: TRACE
(TRACE calculates the optics in presence of space charge)
These lengths are copied from the old MAD8 files. It should be studied if they should still be used.
The element files with the correct length are also available.

----------------------------------------------------------------------------------

There are one type of optics: Protons at 311 MeV/c (Ekin=50 MeV) - bi.madx

----------------------------------------------------------------------------------

Further information on: http://cern-accelerators-optics.web.cern.ch/cern-accelerators-optics/BI/defaultBI.htm

----------------------------------------------------------------------------------

Excecute the files in the madx directory, with the command:
>madx < bi.madx

----------------------------------------------------------------------------------

The sequence file: lt.seq is updated from measurements by T.Dobers 29/3/2011
